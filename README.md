# Демографический анализ России

## Выполненные задания

### Задание 5 (Ульяна) - Анализ численности населения
- Чтение данных из CSV файла (колонки: год, численность)
- Вывод таблицы с данными за 15 лет
- Построение графика зависимости численности от года
- Расчет максимального процента прироста и убыли населения
- Прогнозирование методом скользящей средней на N лет
- Отображение прогноза на графике другим цветом

### Задание 6 (Вика) - Анализ миграции населения
- Чтение данных из CSV файла (колонки: год, иммигранты, эмигранты)
- Вывод таблицы с данными за 15 лет
- Построение графиков иммигрантов и эмигрантов
- Расчет максимального процента изменения миграционного потока
- Прогнозирование методом скользящей средней на N лет
- Отображение прогноза на графике пунктирной линией
